又一次在年后写年终总结。

工作

2021 年变化有点大，换了工作，一切又重新开始。总体来说对现在的工作还是很满意的，身边的人都很优秀，leader 会关心你成长，做着还算蛮有挑战的事情。

工作上主要方向还是在 CDC 缓存同步，以及一些在线微服务上。设计了一套新的 CDC 同步系统，详细地了解了 MySQL 主从复制协议，也了解了一些流式计算的东西，收获还是挺多的。在线微服务方面因为已经有了比较成熟的方案，所以觉得高并发也不是像大家说的那样难，也有可能我没遇到难的场景。

工作感悟

对于问题应该深度思考，这个在之前思考问题的方式里其实有提到，深度思考的能力，决定了你人生的高度。

要常怀敬畏之心，认真对待每一件事，毕竟哪个技术人员都不想听到用户在网络上发帖说某某 app 又崩了，谁家的技术怎么这么烂这样的话。

充电成果

上一年的计划

  • 每两周输出一篇文章这个事情没有完成，反思一下原因，素材比较难找，工作上遇到的要么别人已经写的很好了，自己在写好像有点复制粘贴的感觉了，要么就是自己要收集大量的资料，时间上又不允许，算了不找借口，本质上还是自己自律能力不行。
  • 代码质量问题持续改进。
  • 算法题偶尔刷。
  • 6.s081 其实年前就把实验完成了，但是输出文章比较晚。

看过的书

  • 《DDIA》已经读了好几遍了。
  • 《Stream system》想接触一下流式计算系统，发现有些地方不是很好读懂，后面可能还得花时间读读。
  • 《Kafka: The Definitive Guide, 2nd Edition》Kafka 权威指南里面介绍了 Kafka 的一些运行机制，但是感觉还是不够深入。
  • 《High Performance MySQL》这本书也是 MySQL 广受好评的一本书，但是对于深入了解 MySQL 来说更推荐 MySQL Internals Manual 。
  • 《原则》形成自己的原则，做事情的时候就会更理性。
  • 《刷新：重新发现商业与未来》大部分章节像是微软的广告，不过前面几节作者的经历能让我们深受启发，比如同理心，对待有先天性缺陷孩子时作者的态度：生活中的问题并不总是能以我们所希望的方式解决。相反，我们要学会面对。
  • 《习惯的力量》学会自律。
  • 《人类简史》人类文明的变迁。
  • 《论中国经济》传奇经济学家林毅夫对中国过去经济发展模式的论述，认为未来15年，中国有可能实现每年5%或6%增长，我们拭目以待？

计划

工作

  • 减少碎片化时间, 专注聚焦

  • 多分享交流，注重人与人之间的连接

  • 深度思考，尽可能把事情做到极致

充电计划

  • 每日深度阅读时间不少于一小时

  • 坚持刷算法题

  • 实现一个简单的数据库

  • 坚持博客输出

生活

  • 规律作息时间

  • 开始运动减肥
